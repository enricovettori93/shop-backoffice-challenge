## Installazione
- Prerequisito: avere installato NVM nel sistema operativo
- Creare un file .env a partire da .env.example
- run `nvm use`
- run `npm install`
- run `npm run dev`

Da qui verrá eseguito il webserver con vite e sará raggiungibile all'indirizzo http://localhost:5173/
